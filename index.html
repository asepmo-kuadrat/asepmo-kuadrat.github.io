Panduan Lengkap Membuat Website Gratis dengan GitHub Pages

Ditulis oleh Dev Assistant

25 Mei 2026

3 menit membaca

Memiliki website portofolio atau blog pribadi tidak harus mahal. GitHub Pages adalah solusi luar biasa yang memungkinkan Anda meng-hosting website statis secara gratis langsung dari repositori GitHub. Artikel ini akan membahas langkah-langkah praktis untuk membangun dan meluncurkan website Anda sendiri dalam hitungan menit.

Mengapa Memilih GitHub Pages?

Sebelum memulai, mari kita lihat perbandingan kelebihan dan kekurangan menggunakan layanan ini untuk mempertimbangkan apakah GitHub Pages adalah pilihan yang tepat untuk Anda:

Kelebihan Kekurangan
Hosting sepenuhnya gratis tanpa biaya bulanan Hanya mendukung website statis (HTML, CSS, JS)
Mendapatkan nama domain bawaan (username.github.io) Tidak mendukung database atau bahasa backend (PHP, Python)
Terintegrasi langsung dengan sistem version control Git Terdapat batasan kapasitas penyimpanan dan bandwidth
Mendukung penggunaan domain kustom secara mudah Kurang cocok untuk website e-commerce kompleks

Persiapan Awal

Pastikan Anda telah memenuhi beberapa persyaratan dasar berikut sebelum lanjut ke tahap pembuatan:

  • Memiliki akun GitHub yang aktif.
  • Menyiapkan file website statis dasar (minimal satu buah file HTML).
  • Memiliki pemahaman dasar tentang manajemen file.

Langkah-Langkah Pembuatan Website

Proses peluncuran website melalui antarmuka web GitHub sangat sederhana. Ikuti langkah-langkah di bawah ini:

  1. Buat Repositori Baru: Klik tombol "New" pada halaman utama repositori di akun GitHub Anda.
  2. Beri Nama Spesifik: Isi kolom Repository name dengan format wajib username.github.io (ganti "username" dengan nama akun Anda yang sebenarnya).
  3. Pilih Akses Publik: Pilih opsi "Public" agar website Anda dapat diakses oleh siapa saja di internet.
  4. Selesaikan Pembuatan: Gulir ke bawah dan klik tombol "Create repository".
  5. Unggah File Website: Pada halaman repositori yang baru jadi, klik "Add file" lalu pilih "Upload files".
  6. Pastikan Ada File Index: Unggah file website Anda dan pastikan halaman utama diberi nama index.html.
  7. Simpan Perubahan: Tulis deskripsi singkat pada kolom yang tersedia, lalu klik tombol "Commit changes".
  8. Akses Website: Buka browser web Anda dan ketikkan alamat https://username.github.io.

Kesimpulan

Website Anda sekarang sudah live dan bisa dilihat oleh seluruh dunia! GitHub Pages menawarkan cara yang sangat efisien, cepat, dan hemat biaya untuk menampilkan karya Anda secara online. Dengan menguasai dasar-dasar ini, Anda dapat mulai membangun portofolio digital yang profesional tanpa harus menyewa server yang mahal.
