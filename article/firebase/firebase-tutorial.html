Mengenal Apa Itu Firebase

Firebase adalah platform Backend-as-a-Service (BaaS) dari Google untuk mempercepat pembangunan aplikasi modern tanpa memusingkan pengelolaan server.

Autentikasi Aman 🔐

Firebase Authentication menyediakan sistem sign-in aman, mendukung pendaftaran via email/sandi, integrasi akun sosial (Google, Apple, Facebook), hingga verifikasi SMS secara out-of-the-box.

Cloud Firestore 🗄️

Database dokumen fleksibel, berskala global, dan real-time. Memungkinkan penyimpanan profil dan data dinamis terstruktur yang disinkronkan secara mulus ke jutaan pengguna dalam milidetik.

Notifikasi FCM 🔔

Firebase Cloud Messaging (FCM) mengizinkan developer mengirimkan pesan notifikasi push secara instan ke browser web atau perangkat mobile (Android/iOS) sepenuhnya gratis.

Setup Proyek Multiplatform

Platform Android

Integrasi di Android Studio

  1. Daftarkan nama paket (*Package Name*) aplikasi Anda ke Firebase Console.
  2. Unduh file konfigurasi google-services.json dan simpan ke dalam direktori app/ proyek Anda.
  3. Tambahkan plugin Google Services pada file build.gradle (level proyek).
  4. Tambahkan SDK dependensi Firebase yang diinginkan di file build.gradle (level modul/app).
  5. Klik **"Sync Now"** untuk menyelesaikan instalasi paket gradle.
Platform Web (HTML & JS)

Integrasi Web Dasar

  1. Buat aplikasi web baru di Firebase Console, lalu salin modul konfigurasi objek JSON yang diberikan.
  2. Gunakan tag skrip dengan tipe modul (type="module") agar dapat mengimpor library resmi secara bersih.
  3. Hubungkan file konfigurasi tersebut dengan memanggil method initializeApp(firebaseConfig).
  4. Konfigurasikan database dan rules autentikasi di dalam konsol sebelum melakukan pengujian pengiriman data.

				  const app = initializeApp(firebaseConfig);

				  const db = getFirestore(app);
				  const messaging = getMessaging(app);

				  // 🔍 3. Mengambil Referensi Elemen Antarmuka HTML
				  const registerContainer = document.getElementById('register-container');
				  const loginContainer = document.getElementById('login-container');
				  const profileContainer = document.getElementById('profile-container');

				  const loginForm = document.getElementById('login-form');
				  const profileForm = document.getElementById('profile-form');
				  const logoutBtn = document.getElementById('logout-btn');

				  // ==========================================
				  // 🔐 INTEGRASI AUTENTIKASI (LOGIN & DAFTAR)
				  // ==========================================

				  // Logika Pendaftaran Akun Baru
				  registerForm.addEventListener('submit', (e) => {
				  e.preventDefault();
				  const email = document.getElementById('reg-email').value;
				  const password = document.getElementById('reg-password').value;

				  createUserWithEmailAndPassword(auth, email, password)
				  .then((userCredential) => {
				  alert(`Pendaftaran berhasil! UID: ${userCredential.user.uid}`);
				  registerForm.reset();
				  })
				  .catch((error) => alert(`Gagal daftar: ${error.message}`));
				  });

				  // Logika Masuk Akun
				  loginForm.addEventListener('submit', (e) => {
				  e.preventDefault();
				  const email = document.getElementById('login-email').value;
				  const password = document.getElementById('login-password').value;

				  signInWithEmailAndPassword(auth, email, password)
				  .then((userCredential) => {
				  alert("Selamat datang kembali!");
				  loginForm.reset();
				  })
				  .catch((error) => alert(`Gagal masuk: ${error.message}`));
				  });

				  // Logika Keluar Akun (Logout)
				  logoutBtn.addEventListener('click', () => {
				  signOut(auth)
				  .then(() => alert("Anda telah keluar."))
				  .catch((error) => alert(`Error: ${error.message}`));
				  });

				  // ==========================================
				  // 🕵️ PEMANTAU STATUS REAL-TIME (AUTH STATE)
				  // ==========================================
				  onAuthStateChanged(auth, (user) => {
				  if (user) {
				  // Pengguna Login: Update DOM Kontainer
				  registerContainer.style.display = 'none';
				  loginContainer.style.display = 'none';
				  profileContainer.style.display = 'block';
				  userEmailDisplay.textContent = user.email;

				  // Ambil Data Profil dari Firestore secara otomatis
				  const docRef = doc(db, "users", user.uid);
				  getDoc(docRef)
				  .then((docSnap) => {
				  if (docSnap.exists()) {
				  const data = docSnap.data();
				  document.getElementById('profile-name').value = data.namaLengkap || '';
				  document.getElementById('profile-phone').value = data.nomorTelepon || '';
				  }
				  })
				  .catch((err) => console.error("Gagal ambil profil:", err));

				  // Aktifkan Notifikasi FCM
				  aktifkanNotifikasi(user);
				  } else {
				  // Pengguna Keluar: Reset DOM Kontainer ke Tampilan Publik
				  registerContainer.style.display = 'block';
				  loginContainer.style.display = 'block';
				  profileContainer.style.display = 'none';
				  }
				  });

				  // ==========================================
				  // 💾 DATABASE FIRESTORE (UPDATE PROFIL)
				  // ==========================================
				  profileForm.addEventListener('submit', (e) => {
				  e.preventDefault();
				  const user = auth.currentUser;
				  if (!user) return;

				  const name = document.getElementById('profile-name').value;
				  const phone = document.getElementById('profile-phone').value;

				  setDoc(doc(db, "users", user.uid), {
				  namaLengkap: name,
				  nomorTelepon: phone,
				  email: user.email
				  }, { merge: true })
				  .then(() => alert("Profil berhasil diperbarui di Firestore!"))
				  .catch((err) => alert(`Gagal simpan: ${err.message}`));
				  });

				  // ==========================================
				  // 🔔 FIREBASE CLOUD MESSAGING (FCM)
				  // ==========================================
				  function aktifkanNotifikasi(user) {
				  Notification.requestPermission().then((permission) => {
				  if (permission === 'granted') {
				  // Dapatkan Token FCM Anda (Ganti dengan Kunci VAPID dari Firebase Console)
				  getToken(messaging, { vapidKey: 'YOUR_VAPID_KEY_HERE' })
				  .then((currentToken) => {
				  if (currentToken) {
				  console.log("Token FCM Pengguna:", currentToken);
				  // Simpan token ke Firestore untuk digunakan saat pengiriman notifikasi terpusat
				  setDoc(doc(db, "users", user.uid), { fcmToken: currentToken }, { merge: true });
				  }
				  })
				  .catch((err) => console.error('Kesalahan Token FCM:', err));
				  }
				  });
				  }
