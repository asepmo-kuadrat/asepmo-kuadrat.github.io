Apa itu Firebase?

Firebase adalah platform Backend-as-a-Service (BaaS) yang dikembangkan oleh Google. Ini memberikan solusi atas kerumitan membangun sistem backend sendiri dari awal.

Kecepatan

Tidak perlu menulis kode backend yang panjang. Fitur tersedia dan terintegrasi dengan cepat.

Skalabilitas

Otomatis membesarkan kapasitasnya saat pengguna melonjak tanpa takut server down.

Real-time Data

Perubahan data di server langsung tersinkronisasi ke semua pengguna seketika.

Ekosistem Lengkap

Analitik, database, penyimpanan file, dan hosting tersedia di satu tempat.

Firebase Hosting (Domain firebaseapp.com)

Firebase Hosting memungkinkan Anda mengunggah website statis atau dinamis secara gratis dengan SSL/HTTPS terpasang otomatis.

Terminal Commands

# 1. Instal Firebase CLI secara global

npm install -g firebase-tools

# 2. Login ke akun Google Anda

firebase login

# 3. Inisialisasi proyek di dalam folder website Anda

firebase init hosting

# 4. Unggah website ke internet!

firebase deploy

Autentikasi: Login & Register

Menangani sistem login sangat mudah menggunakan Firebase Auth. Mendukung kombinasi Email/Password dan Single Sign-On (SSO) seperti Google. 

import { initializeApp } from "https://www.gstatic.com/firebasejs/10.12.0/firebase-app.js";
				import { getAuth, createUserWithEmailAndPassword, signInWithEmailAndPassword, GoogleAuthProvider, signInWithPopup } from "https://www.gstatic.com/firebasejs/10.12.0/firebase-auth.js";

				const app = initializeApp({ /* Konfigurasi Firebase Anda */ });
				const auth = getAuth(app);
				const provider = new GoogleAuthProvider();

				// Logika Login Email
				signInWithEmailAndPassword(auth, email, password)
				.then(user => console.log("Login Berhasil!"))
				.catch(err => console.error(err));

				// Logika Login Google
				signInWithPopup(auth, provider)
				.then(result => console.log("Halo, " + result.user.displayName));

Firebase Storage & Update Profil

Firebase Storage digunakan untuk menyimpan berkas fisik, sangat cocok dipadukan dengan `updateProfile` untuk memperbarui foto pengguna. 

import { getStorage, ref, uploadBytes, getDownloadURL } from "https://www.gstatic.com/firebasejs/10.12.0/firebase-storage.js";
				import { updateProfile } from "https://www.gstatic.com/firebasejs/10.12.0/firebase-auth.js";

				const storage = getStorage(app);

				// Fungsi Mengunggah Gambar
				async function uploadFoto(file, user) {
				const fileRef = ref(storage, `foto_profil/${user.uid}`);

				// 1. Unggah file ke Storage
				await uploadBytes(fileRef, file);

				// 2. Dapatkan URL publik gambar
				const url = await getDownloadURL(fileRef);

				// 3. Simpan URL ke profil Auth
				await updateProfile(user, { photoURL: url });
				}

Cloud Firestore

Database NoSQL modern yang membagi data menjadi Koleksi dan Dokumen. Sempurna untuk menyimpan biodata, keranjang belanja, atau chat. 

import { getFirestore, doc, setDoc, getDoc } from "https://www.gstatic.com/firebasejs/10.12.0/firebase-firestore.js";

				const db = getFirestore(app);

				// Menyimpan Biodata
				async function simpanBiodata(uid, teksBio) {
				const docRef = doc(db, "users", uid); // Koleksi 'users', Dokumen 'UID'
				await setDoc(docRef, { biodata: teksBio }, { merge: true });
				}

				// Membaca Biodata
				async function ambilBiodata(uid) {
				const docRef = doc(db, "users", uid);
				const hasil = await getDoc(docRef);

				if (hasil.exists()) {
				console.log(hasil.data().biodata);
				}
				}

Bonus: Node.js di Termux

Node.js membawa JavaScript ke sisi server. Anda bisa menginstalnya langsung di HP Android menggunakan emulator terminal Termux.

1. Instalasi Node.js

# Update sistem

pkg update && pkg upgrade

# Instal Node.js dan Text Editor

pkg install nodejs nano

2. Membuat Web Server (server.js)

const http = require('http');

					const server = http.createServer((req, res) => {
					res.statusCode = 200;
					res.setHeader('Content-Type', 'text/html');
					res.end('<h1>Halo dari Termux!</h1>');
					});

					server.listen(3000, '127.0.0.1', () => {
					console.log('Server berjalan!');
					});

# Jalankan server dan buka browser di localhost:3000

node server.js
