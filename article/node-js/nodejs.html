Kuasai Pengembangan Sisi Server dengan NodeJS

Selamat datang di portal panduan lengkap pemrograman Node.js. Dari dasar teori, penggunaan di HP (Termux), integrasi Tailwind CSS, hingga simulasi database real-time MongoDB Atlas.

1

Mengenal Node.js: Revolusi JavaScript di Sisi Server

Apa Itu Node.js?

Node.js adalah sebuah runtime environment (lingkungan eksekusi) untuk JavaScript. Sebelum ada Node.js, JavaScript hanya bisa berjalan di dalam web browser (seperti Google Chrome atau Firefox). Node.js hadir mendobrak batasan tersebut dan membiarkan JavaScript berjalan langsung di komputer atau server Anda.

Dengan ini, JavaScript resmi menjadi bahasa pemrograman serbaguna yang bisa digunakan untuk membangun aplikasi backend (*sisi server*), memanipulasi file, mengelola database, serta merakit *CLI tools*.

Fakta Kunci Node.js:

  • Diciptakan oleh Ryan Dahl pada tanggal 27 Mei 2009.
  • Diperkenalkan pertama kali di acara JSConf EU di Berlin, Jerman.
  • Ditenagai oleh Mesin V8 Google Chrome (ditulis dalam bahasa C++).

Mengapa Diciptakan?

Ryan Dahl gelisah melihat arsitektur web server lama (seperti Apache) yang memboroskan memori dengan membuat satu thread baru untuk setiap koneksi tunggal. Ia terinspirasi setelah melihat *progress bar* unggahan Flickr yang tidak efisien.

Dahl merancang Node.js dengan konsep Single-Threaded namun menggunakan sistem Asynchronous (Non-Blocking I/O) lewat pustaka libuv, memungkinkan satu thread melayani puluhan ribu koneksi secara simultan.

Arsitektur: Event-Driven & Non-Blocking
2

Membuat Web Server NodeJS di Termux (Android)

Anda tidak membutuhkan komputer berspesifikasi dewa untuk memulai petualangan backend. Dengan menggunakan aplikasi emulator terminal **Termux** di HP Android, Anda dapat menyulap gawai genggam Anda menjadi mini-server NodeJS.

Langkah 1: Setup Lingkungan di Termux

# 1. Update repositori sistem
pkg update && pkg upgrade -y

# 2. Install NodeJS dan Text Editor Nano
pkg install nodejs nano -y

# 3. Buat direktori kerja
mkdir server-termux && cd server-termux

Langkah 2: Menulis File Server (`server.js`)

Gunakan editor dengan perintah nano server.js dan isi file dengan skrip dasar di bawah:

const http = require('http');

				  const hostname = '127.0.0.1';
				  const port = 3000;

				  const server = http.createServer((req, res) => {
				  res.statusCode = 200;
				  res.setHeader('Content-Type', 'text/html');
				  res.end('<h1>Halo dari Node.js di Termux!</h1>');
				  });

				  server.listen(port, hostname, () => {
				  console.log(`Server berjalan di http://\${hostname}:\${port}/`);
				  });

Jalankan dengan perintah node server.js lalu buka peramban di HP dan arahkan ke alamat http://localhost:3000.

3

Integrasi Tailwind CSS Menggunakan Node.js

Di ekosistem modern, NodeJS tidak hanya melayani lalu lintas *backend*, melainkan juga bertindak sebagai mesin pembangun (*build tools*) handal untuk frontend Anda, seperti mengompilasi **Tailwind CSS**.

1. Instalasi Tailwind CLI

# Inisialisasi npm
npm init -y

# Pasang Tailwind CSS & CLI
npm install tailwindcss @tailwindcss/cli

2. Konfigurasi Input CSS

Buat file di src/input.css dengan baris berikut:

@import "tailwindcss";

3. Menjalankan Proses Compile (Build)

Jalankan perintah berikut agar Tailwind mendeteksi kelas CSS yang Anda gunakan dan mengompilasinya:

npx @tailwindcss/cli -i ./src/input.css -o ./src/output.css --watch
4

Membangun API CRUD dengan MongoDB Atlas Gratis

Langkah puncak dari pembelajaran NodeJS adalah merangkai server RESTful API yang mendukung proses **CRUD** dan menyimpannya ke awan menggunakan basis data NoSQL gratis dari **MongoDB Atlas**.

Kenapa MongoDB Atlas?

MongoDB Atlas menawarkan klaster server database gratis seumur hidup sebesar 512 MB. Kapasitas ini sangat ideal untuk pengujian API, belajar struktur data non-relasional, maupun meluncurkan purwarupa (*MVP*) aplikasi Anda.

Arsitektur Kode Express + Mongoose

Berikut adalah kode inti server Express kita untuk melakukan manajemen produk:

const express = require('express');
				  const mongoose = require('mongoose');
				  require('dotenv').config();

				  const app = express();
				  app.use(express.json());

				  // Hubungkan MongoDB Atlas
				  mongoose.connect(process.env.MONGO_URI)
				  .then(() => console.log('Connected to DB!'));

				  // Schema Produk
				  const productSchema = new mongoose.Schema({
				  nama: { type: String, required: true },
				  harga: { type: Number, required: true },
				  stok: { type: Number, default: 0 }
				  });
				  const Product = mongoose.model('Product', productSchema);

				  // [CREATE] Tambah Data
				  app.post('/api/products', async (req, res) => {
				  const product = new Product(req.body);
				  await product.save();
				  res.status(210).json(product);
				  });

				  // [READ] Tampilkan Semua
				  app.get('/api/products', async (req, res) => {
				  const products = await Product.find();
				  res.json(products);
				  });

				  // [UPDATE] Edit Data
				  app.put('/api/products/:id', async (req, res) => {
				  const updated = await Product.findByIdAndUpdate(req.params.id, req.body, {new:true});
				  res.json(updated);
				  });

				  // [DELETE] Hapus Data
				  app.delete('/api/products/:id', async (req, res) => {
				  await Product.findByIdAndDelete(req.params.id);
				  res.json({ message: 'Data terhapus!' });
				  });

				  app.listen(3000, () => console.log('Server runs on port 3000'));
